Important FAQ

What is the document object model (DOM) ?

The Document Object Model (DOM) is a programming interface that represents HTML and XML documents as a tree of objects, where each node corresponds to parts of the document (elements, attributes, text). It allows developers to dynamically manipulate the document's structure, content, and style using languages like JavaScript. The DOM facilitates interactive web experiences by enabling tasks such as adding, removing, or modifying elements and handling events triggered by user actions. This makes it a crucial component in modern web development, allowing for real-time updates and user interactions on web pages.

How do you select an element from DOM ?

To select an element from the DOM, you can use various methods in JavaScript. The most common are:

  • document.getElementById('id') for selecting by ID.
  • document.getElementsByClassName('class') for selecting by class name.
  • document.getElementsByTagName('tag') for selecting by tag name.
  • document.querySelector('selector') for selecting the first matching element using CSS selectors.
  • document.querySelectorAll('selector') for selecting all matching elements.

What is event delegation in the context of DOM and why is it usefull?

Event delegation is a technique in the DOM that involves attaching a single event listener to a parent element instead of multiple listeners to child elements. When an event occurs on a child element, the event bubbles up to the parent, where it can be handled. This approach is useful for several reasons: it reduces memory usage by minimizing the number of event listeners, simplifies code maintenance, and allows for dynamic content handling, as new child elements will automatically inherit the event listener without needing additional setup. Overall, it enhances performance and efficiency in event management.

How can you change an element’s content?

Use element.textContent or element.innerHTML.

What does querySelector do?

The `querySelector` method in the DOM selects the first element that matches a specified CSS selector. It allows for flexible selection using class names, IDs, attributes, or complex selectors, returning the element as an object. If no matching element is found, it returns `null`..

How can you add a new element to the DOM?

  • Create the Element: Use document.createElement('tagName') to create a new element, where 'tagName' is the desired HTML tag (e.g., 'div', 'p').
  • Set Attributes or Content: Modify the new element’s properties, such as element.textContent or element.setAttribute('attr', 'value').
  • Select a Parent Element: Use a method like document.getElementById('parentId') to select the parent element where you want to append the new element..
  • Append the Element: Use parentElement.appendChild(newElement) to add the new element to the DOM.
